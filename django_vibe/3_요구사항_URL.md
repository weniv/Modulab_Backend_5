# ICT 교육 동영상 스트리밍 플랫폼 - URL 구조 설계 요구사항

## 1. 프로젝트 개요

### 1.1 서비스 설명
ICT 분야 교육을 위한 온라인 동영상 스트리밍 학습 플랫폼을 구축한다.
- **레퍼런스 서비스**: Udemy, 인프런, 코세라

### 1.2 기술 스택 (권장)
- **Backend**: Django 모놀리식

## 2. 핵심 기능 요구사항

### 2.1 사용자 인증
| 기능 | 설명 | 우선순위 |
|------|------|----------|
| 회원가입 | 이메일/비밀번호 기반 회원가입 | 필수 |
| 로그인 | 이메일/비밀번호 로그인 | 필수 |
| 로그아웃 | 세션 종료 및 토큰 무효화 | 필수 |

### 2.2 강의 관련 기능
| 기능 | 설명 | 우선순위 |
|------|------|----------|
| 강의 목록 조회 | 전체 강의 목록 페이지네이션 | 필수 |
| 강의 검색 | 키워드, 카테고리별 검색 | 필수 |
| 강의 상세 보기 | 강의 정보, 커리큘럼, 리뷰 표시 | 필수 |
| 강의 수강 신청 | 강의 구매/등록 처리 | 필수 |
| 동영상 재생 | 스트리밍 방식 영상 재생 | 필수 |
| 학습 진도 관리 | 시청 기록 저장 및 이어보기 | 필수 |

### 2.3 사용자 대시보드
| 기능 | 설명 | 우선순위 |
|------|------|----------|
| 내 강의 목록 | 수강 중인 강의 리스트 | 필수 |
| 프로필 관리 | 사용자 정보 수정 | 필수 |
